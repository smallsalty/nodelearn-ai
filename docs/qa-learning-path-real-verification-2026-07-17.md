- 个性化学习路径生成：`PASS_REAL`。路径由真实 DeepSeek 生成中文标题、说明、阶段和任务标题，并在安全审核通过后返回；页面没有显示模拟路径文案、内部节点 ID 或英文任务类型。

## 本次实现

- 将界面名称“对话学习”统一为“问答助手”，页面只保留“开始问答”和“问答历史”。
- 删除问答页的知识节点展示面板和引用步骤面板；课程节点仍作为请求上下文，不再作为重复的页面流程展示。
- 问答助手页面与学习侧栏共用 `ChatSession`、`ChatMessage` 和历史读取逻辑；回答完成后刷新同一份历史。
- 普通问答固定执行课程检索、DeepSeek 回答、安全审核、PostgreSQL 持久化，不在真实模式失败时回退模拟回答。
- 显式复用会话时先校验会话存在且归属于请求用户，其他用户的会话不会进入模型提示词。
- 学生画像将内部枚举、课程编号和已知节点编号转换为中文展示。
- 学习路径真实模式调用 DeepSeek 规划，弱项节点、用户目标中明确提到的知识点及必要前置节点共同组成候选任务；任务写入逐日建议完成时间。
- 每个路径任务提供三种学习工具建议、中文用途说明、可直接复制的中文提示词和对应入口。
- 学习路径生成前端超时单独调整为 3 分钟，以覆盖真实模型规划耗时，不改变其他接口的默认超时。
